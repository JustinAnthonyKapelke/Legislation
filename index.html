Security by Design

  • Plan threats before writing code
  • Threat modeling during design phase
  • Security can't be added after, nya!
  • Document all decisions carefully

Vulnerability Management

  • Active exploit = 24h early warning
  • Full report required in 72 hours
  • "Fix it later" is actually illegal 😿
  • Coordinated disclosure required

Secure by Default

  • Ship safe out of the box, please!
  • Zero weak default passwords 🙅
  • All security features on by default
  • No manual hardening needed by users

Dependency Management

  • SBOM for every single library
  • Auto-scan for known risks daily
  • Log4j? CRA was literally made for this
  • Know your full supply chain!

  • Document every mitigation taken
  • Risk management never stops 🌸

Technical Documentation

  • Audit-ready evidence always available
  • Log every design decision made
  • Maintained risk register required
  • CE marking declaration of conformity
